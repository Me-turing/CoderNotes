## 最近的文件和位置

`Ctrl + E` 查看最近使用的文件

`Ctrl + Shift + E `查看最近使用的位置

## 上一个/下一个匹配

`Ctrl + F` 在当前文件中查找

`Enter` 或 `F3 `切换到下一个匹配

`Shift + Enter`  或 `Shift + F3` 切换到上一个匹配
